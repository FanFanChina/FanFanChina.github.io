使用 Hexo+Github Pages 搭建静态博客教程

发表于2023-06-13|更新于2023-12-26|Tutorial
|阅读量:

引言

本文阐述如何使用 Hexo + Github Pages搭建个人博客

本地安装 Hexo

  1. 官方文档Hexo
  2. 安装git、node
  3. 选着一个存放博客资源的文件夹在给文件夹下打开 command prompt 输入命令 init Hexo
  1. 安装 npm 包管理器
  1. 安装hexo的上传插件deploy-git
  1. 本地测试Hexo服务
  1. 选择一个自己喜欢的主题，如何配置见相应主题配置文档

Github Pages

  1. Github 提供了一个免费的静态页面托管服务 Github Pages
  2. Github Pages 其实是一个Github Repository, 仓库配置如下
  1. 为了避免每次hexo deploy到github时输入密码，在hexo站点配置中
  1. 配置本机与github的ssh免密连接（略）
  2. 站点和主题配置完毕以后，上传到github
  1. 打开网站 your_user_name.github.io 即可

自定义域名

  1. 购买域名，例如本站域名 onism.cn

  2. 配置域名解析服务

    1. 打开command prompt并ping自己的博客地址获取 IP

    2. 添加A记录、主机记录为@、记录值为IP & 添加CNAME记录、主机记录为www、记录值为 fanfanchina.github.io

  3. 配置github pages

    1. 打开github找到自己的托管静态页面的仓库
    2. 点击settings -> pages -> custom domain 输入 www.onism.cn

  4. 在本地博客文件source文件夹下添加无后缀的文件CNAME，文件内容为自己的域名 www.onism.cn (这一步很重要，如果没有添加CNAME文件每一次hexo d上传后自定义域名配置会失效!!!)
