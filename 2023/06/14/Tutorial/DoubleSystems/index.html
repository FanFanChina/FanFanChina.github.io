如何搭建 Windows 11 + Ubuntu 22.04 双系统

发表于2023-06-14|更新于2023-12-26|Tutorial
|阅读量:

备份文件（目录树）

一般而言，需要备份的文件类型如下：

  • 文档：word、ppt、excel、pdf、markdown 等
  • 项目：Python项目、C++项目、Golang项目等
  • 媒体文件：图片、音乐、视频等
  • 环境文件：MingW64、Anaconda 等
  • 必要软件：Clash、Bindizip

备份恢复时，我们需要重构目录结构，此时我们可以用python代码自动化这个过程

制定分区策略

我有两个 512 GB 的固态，可以把两个系统装在其中一个固态中，剩下一个固态作为公用盘

对于装系统的 512 GB 固态分区方案如下：

Windows Ubuntu
EFI 系统引导区（FAT32） 500 MB EFI 系统引导区（FAT32）500 MB
C盘（NTFS） 120 GB Swap 分区（SWAP） 20 GB
D盘（NTFS） 197 GB 根目录 / 挂载分区（EXT4） 47 GB
/home 目录挂载分区（EXT4） 92 GB

剩下一个512固态格式化为NTFS分区即可

制作 PE 系统

微PE工具箱下载

安装 Windows

下载 Win11系统镜像

进入PE工具，点击 Windows 安装工具，按规划好的分区安装 Windows 系统即可

安装 Ubuntu

进入安装好的 Windows 系统，下载 Ubuntu 桌面版镜像，下载 Win32 Disk Imager 工具，将 Ubuntu 镜像写入U盘。

将系统锁屏，按住Shift并点击重启按钮，重启后点击UEFI模式，按规划好的分区安装 Ubuntu 镜像
