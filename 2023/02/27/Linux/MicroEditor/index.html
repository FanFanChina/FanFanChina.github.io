MicroEditor 使用教程

发表于2023-02-27|更新于2023-12-26|Linux
|阅读量:

简介

Micro Editor is a modern and intuitive terminal-based text editor.

安装

  1. Micro Editor Office Website
  2. Change Directory 到你想要的下载软件的位置，然后打开Terminal输入命令
  3. 下载完成后你会得到一个binary file – micro，输入 ./micro可以直接使用micro editor 编辑器

使用

注册为系统编辑器

  1. 授予二进制文件micro root权限
  2. 将 micro 文件移动到/usr/local/bin文件夹下（其他你喜欢的文件夹也可以）
  3. 然后你就能在任意位置使用micro + （文件名称）使用micro了

基本快捷键

  • 退出 – control + q
  • 保存 – control + s
  • 命令行界面 – control + e
  • 切换面板 – control + w

如果你想看具体的快捷键功能，你可以在命令行模式输入以下命令

开启剪切板功能

  • 复制 – control + c
  • 粘贴 – control + v
  • 剪切 – control + x
  • 选中多行 – shift + 方向键

Command Bar 命令

  • Replace All Same Word
  • Vertical Split Screen
  • Horizon Split Screen
  • Set Color Sheme （如果不知道主题名称可以用tab补全、cmc-16比较好看）
