字符参与加法运算

  • char 类型的数据通过 ASCII 先转化为 int，再进行运算

字符串参与加法运算

  • 当＂+”操作中出现字符串时，这个“+”是字符串连接符，而不是算术运算符了
  • 会将前后的数据进行拼接，并产生一个新的字符串
  • 连续进行”+“操作时，从左到右逐个执行

自增 or 自减运算符

  • 无论自增 or 自减运算符放到前面还是后面，效果都是实现自增 or 自减
  • 当携带自增 or 自减运算符的同时进行运算或输出时，运算符在后面的从本位开始，在前面的从下一位开始

左移运算符：向左移动，低位补 0

右移运算符：向右移动，高位补 0 或 1
