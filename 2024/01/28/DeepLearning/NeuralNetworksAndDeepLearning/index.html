Logistic Regression

Sigmoid 函数是一种常用的数学函数，也被称为 Logistic 函数

$\sigma(x) = \dfrac{1}{1 + e^{-x}}$

$\lim_{x \to + \infin}\sigma(x) = 1$

$\lim_{x \to - \infin}\sigma(x) = 0$

Sigmoid 函数的图形呈 S 形状，这也是它被称为 Sigmoid 的原因，它能够将任意实数映射到一个范围在 0 到 1 之间的值。这个函数在机器学习和神经网络中经常用于产生二进制输出，例如在二元分类问题中，Sigmoid 函数的输出可以被解释为概率值，表示某个样本属于正类别的概率。

Logistic Regression Gradient Descent

对于每一个样本：

$L(a, y) = -(ylog(a) + (1 - y)log(1 - a))$

$\dfrac{dL}{da} = \dfrac{y - 1}{a - 1} - \dfrac{y}{a}$

$a = \dfrac{1}{1 + e^{-Z}} = \dfrac{e^Z}{e^Z + 1} = 1 - \dfrac{1}{e^Z + 1} = 1 - (e^Z + 1)^{-1}$

$\dfrac{da}{dZ} = (-1)(-1)(e^Z + 1)^{-2}e^Z = (e^Z + 1)^{-2}e^Z = (e^Z + 1)^{-2}(e^{2Z}e^{-2Z})e^Z = [(e^Z + 1)e^{-Z}]^{-2}e^{-2Z}e^Z = (1 + e^{-Z})^{-2}e^{-2Z}e^Z$

$= (1 + e^{-Z})^{-1}\dfrac{e^{-Z}}{1 + e^{-Z}} = \dfrac{1}{1 + e^{-Z}}(1 - \dfrac{1}{1 + e^{-Z}}) = a(1 - a)$

$\dfrac{dL}{dZ} = \dfrac{dL}{da}\dfrac{da}{dZ} = a(1 - y) - y(1 - a) = a - y$

$\dfrac{dL}{dZ} = a - y$，简写为 $dZ = a - y$

同理可得：

$\dfrac{dL}{dw_1} = \dfrac{dL}{dZ}\dfrac{dZ}{dw_1} = x_1dZ = x_1(a - y)$

$\dfrac{dL}{db} = \dfrac{dL}{dZ}\dfrac{dZ}{db} = dZ = a - y$
